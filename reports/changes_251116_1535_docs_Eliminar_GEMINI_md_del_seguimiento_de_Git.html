El cambio actual consiste en dejar de rastrear el fichero GEMINI.md en el repositorio de Git, manteniendo una copia local para el usuario.

1. Eliminación de GEMINI.md del Repositorio

  • Qué se hizo: Se ha eliminado el archivo GEMINI.md del seguimiento de Git.
  • Por qué: Este archivo contiene directivas internas para el agente de IA que se consideran personales y no deben formar parte del historial del repositorio.
  • Cómo se hizo: Se utilizó el comando git rm --cached GEMINI.md. Este comando elimina el archivo del índice de Git (lo que significa que se borrará del repositorio en el próximo commit) pero lo mantiene intacto en el directorio de trabajo local, ahora como un archivo no rastreado.
